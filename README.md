The 'Graduate CS Student Simulator' folder contains the .exe and the .txt file,
whereas the 'Graduate CS Student Simulator Python' folder contains the .py files (start by opening startmenu.py).

In both cases, if the .txt file is not in the same dir, the game breaks.

(For people who don't know how to use GitHub - Press the green 'Code' button, then press 'Download as ZIP'. Extract the .zip and then play as normal)
